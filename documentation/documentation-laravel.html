Installation

Given below are the steps you need to follow, to install the kannakku_laravel on your system:

Step 1: Open the terminal in your root directory(kannakku_laravel) & to install the composer packages run the following command:

composer install

Step 2: In the root directory, you will find a file named .env.example, rename the given file name to .env and run the following command to generate the key.

php artisan key:generate

Step 3:

  • a)open the xampp,start the apache and mysql
  • b)open the localhost/phpmyadmin in browser and create db_name
  • c)Update the db_name and phpmyadmin table username and password in .env file as per below screenshot

Step 4: Open the terminal in your root directory(kannakku_laravel) & and run the following command for migrate the table:

php artisan migrate

Step 5: Now you can run this Template in any web browser from your PC or Laptop.

LOGIN ROUTE: 
For example: localhost/kannakku_laravel/blue/public/login or www.example.com/kannakku_laravel/blue/public/login
REGISTER ROUTE: 
For example: localhost/kannakku_laravel/blue/public/registration or www.example.com/kannakku_laravel/blue/public/registration
DASHBOARD ROUTE: 
For example: localhost/kannakku_laravel/blue/public/dashboard or www.example.com/kannakku_laravel/blue/public/dashboard

  • Composer
